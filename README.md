Portfolio de Asier Rodríguez Lamas

Proyecto de mi portfolio personal donde muestro todos mis proyectos, trayectoria, educación y habilidades como desarrollador de software junior.

Tecnologías utilizadas:
  - React
  - Vite
  - HTML5
  - CSS
  - JavaScript

Como ver el portfolio:
  - Clonar el repositorio de Git mediante "git clone https://github.com/Asierrl1110/miportfolioonline"
  - Instalar dependencias con "npm install"
  - Ejecutar el servidor de desarrollo con "npm run dev"
  - También se puede acceder desde: https://asierrodriguezlamas.vercel.app/
